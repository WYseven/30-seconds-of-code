### last

------------

Returns the last element in an array.

Use `arr.length - 1` to compute the index of the last element of the given array and returning it.

------------

返回数组中的最后一个元素。

使用 `arr.length - 1` 来计算给定数组的最后一个元素的索引并返回。

```js
const last = arr => arr[arr.length - 1];
```

```js
last([1, 2, 3]); // 3
```
